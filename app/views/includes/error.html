Error code: 500

  • 500 - Internal Server Error

    This is a generic error when the server cannot determine the exact problem

  • 501 - Not Implemented

    The server cannot distinguish the request made, or is unable to perform the request

  • 502 - Bad Gateway

    The server is operating as a gateway or proxy server and the upstream server returns an invalid response

  • 503 - Service Unavailable

    The server is unavailable at the present moment, usually due to maintenance or overloading - try again later

  • 504 - Gateway Timeout

    The server is operating as a gateway or proxy server and the upstream server failed to respond in a timely manner
